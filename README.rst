Papis-libgen
============

|Pypi| |Python| |TRAVIS|


Usage
-----

Just hit

::

  papis libgen

Installation
------------

There is a version in PyPi

::

  pip3 install papis-libgen

or

::

  sudo pip3 install papis-libgen
